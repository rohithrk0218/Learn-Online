Selector2

universal selector

it selects all elements on the page.

it indicates with *

it is used to apply styles to all elements.

it is not recommended to use it in large projects.

grouping selector

it groups multiple selectors to apply the same styles.
it is indicated with a comma (,)

combined selector

  1. descendent selector

    2. it selects elements that are descendants of a specified element.

    it is indicated with a space between the selectors.

  2. child selector

    3. it selects elements that are direct children of a specified element.

    it is indicated with a greater than sign (>) between the selectors.

  3. adjacent sibling selector

    4. it selects an element that is immediately preceded by a specified element.

    it is indicated with a plus sign (+) between the selectors.

  4. general sibling selector

    5. it selects all elements that are siblings of a specified element.

    it is indicated with a tilde (~) between the selectors.

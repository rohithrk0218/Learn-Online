Visit Example.com 

    The anchor tag is used to create hyperlinks in HTML.
    It is defined using the <a> element.
    The attributes of the anchor tag are:

        1. href: Specifies the URL of the page the link goes to.
        2. target: Specifies where to open the linked document.

anchor tag consists of 3 states:

  1. Unvisited Link: The link has not been clicked yet.(blue)
  2. Visited Link: The link has been clicked before.(purple)
  3. Active Link: The link is currently being clicked.(red)
